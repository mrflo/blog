You are working on different projects and you can't concentrate on one at a time? This is for you...

July 23 2007, 9:21 PM  by Xavier Damman
I know I'm not the only one who like doing thousand things at one time.
I've always 36 windows opened simultaneously and my workflow is always interrupted by many web IRQ such as twitter, gtalk, msn, email and so on...

This situation results to a productivity drop. The very first symptom being that famous question at the end of the day "What the hell have I done today?".

Months ago I've written a post about a way to improve that situation using a very clean desktop (unicolor wallpaper, no menu bar, and one application at a time using Spirited Away for MacOSX).

I was using this for a while but now I've come up with a brand new idea that I think is even more productive:

Create one user per project.

It's a kiss (Keep It Simple & Stupid) idea and it changed my life (... or not, but at least it made me more productive on my different projects).

Why is that great?
First, it is simple. Even Windows users have this functionality built-in on Windows XP!
Second, it allows you to only see documents, emails, contacts, images, that have a direct link with your project.

Example:
I've now two users on my MacBook: xdamman (the administrator and main one) and KlaxonneMoi, dedicated to the KlaxonneMoi.org project.
When I switch to the latter account my emails, IM contacts, photos (on iPhoto) and so on are only relevant for that project.
So, when I open Firefox, I've only bookmarks and saved loggins and passwords for my xdamman@klaxonnemoi.org accounts (either on gmail, scoopeo, youtube, ...).

When I need to exchange files between my main account and the new one, I just use the drop box (/Users/klaxonnemoi/Public/Drop box). It works fine!

I'm not interrupted by my main contacts (family, friends, ...) when I'm working on the project, and I don't have to deal with private mails and photos and all these stuffs that put my attention away of my workflow.

The only issue is the memory usage. As applications stay open when you switch user, the memory usage increases. Better to have one giga of memory (I've only 756MB and it is limit).
